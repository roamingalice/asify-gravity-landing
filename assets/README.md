# 照片放這裡

## 資料夾用途

```
assets/
  products/   ← 【必要】每件商品一張去背主圖（白底或透明背景）
  models/     ← 【選用】model 穿搭照，翻牌解讀時會用到
```

## 檔名規則

**照著 `slug` 欄命名**，副檔名 `.png`（去背）或 `.jpg`（有背景）。

- 主圖:`products/<slug>.png`
- 穿搭圖:`models/<slug>-model.jpg`

範例:
- `products/ten-black.png`
- `models/ten-black-model.jpg`

## 尺寸建議

- 主圖：**直立 800×1200** 或 **正方 1000×1000**，去背（褲子類建議直立，Bra/上衣正方都可）
- 穿搭圖：**直立 1080×1620** 以上，背景單純（白/米/淺灰最佳）

## 如果撿到「王牌」

15 件裡每次載入會隨機指定 1 件為「今日王牌」（機率 ~8%）。客人撿到 → 跳過正常占卜 → 直接進特殊結局：

> ✦ 妳今天被選中了。
> 今日王牌：**XXX**
> 這是專屬於妳的那一件。

專屬折扣或贈品（細節你定）。

## 分享卡片

占卜結果畫面有「分享」按鈕，會生成一張 1080×1080 的 PNG：
- ASIFY 品牌字樣
- 三張牌 + 關鍵字
- 今日原型名 + 敘事
- 主推商品縮圖
- 短網址或 QR 導回這頁

客人下載後貼 IG / 小紅書 / 限動都可以。
